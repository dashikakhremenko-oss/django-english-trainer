# Английский тренажёр

## Назначение проекта
Веб-приложение для изучения английских слов. Помогает пользователям пополнять словарный запас, повторять слова и проверять свои знания через тесты.

## Инструкция по запуску

1. Клонировать репозиторий:
git clone https://github.com/dashikakhremenko-oss/django-english-trainer.git
cd django-english-trainer

2. Создать виртуальное окружение:
python3 -m venv venv
source venv/bin/activate

3. Установить зависимости:
pip install -r requirements.txt

4. Выполнить миграции:
python3 manage.py migrate

5. Запустить сервер:
python3 manage.py runserver

6. Открыть в браузере:
http://127.0.0.1:8000
